# Ws2019

Example application for https://www.agilemovement.it/workingsoftware/ 2019.


## Installation

Download the repo,

```
mix deps.get
mix compile
iex -S mix
```

Execute the examples that you can find in these two files:

- [console.exs](./console.exs)
- [example_DDD.exs](./example_DDD.exs)
